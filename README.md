# 🌦️ WeatherSphere

WeatherSphere is a modern and responsive weather forecasting web application that provides real-time weather information, detailed weather highlights, a 5-day forecast, interactive temperature charts, and location mapping for any city worldwide.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- ☁️ Current weather conditions
- 💧 Weather highlights
  - Humidity
  - Wind Speed
  - Atmospheric Pressure
  - Visibility
  - UV Index
  - Cloud Coverage
  - Sunrise & Sunset
- 📅 5-Day Weather Forecast
- 📈 Interactive Temperature Trend Chart
- 🗺️ Location Map using Leaflet.js

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Chart.js
- Leaflet.js
